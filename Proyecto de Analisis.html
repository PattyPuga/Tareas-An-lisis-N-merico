Modelo IS-LM

Se compone de ecuaciones estructurales con variables end√≥genas y ex√≥genas, donde algunas ecuaciones son de comportamiento y otras son condiciones de equilibrio o identidades contables. Se considera que el sistema se verifica en cada momento del tiempo t y que el modelo se encuentra en equilibrio est√°tico si los valores de las variables end√≥genas aseguran el vaciado de los mercados.

Supuestos b√°sicos del modelo IS-LM Funci√≥n IS: Y=C(Y‚àíT)+I(r)+G Funci√≥n LM: M/P=L(Y,r)

Con precios fijos: La exposici√≥n convencional del modelo IS-LM plantea una divisi√≥n entre una econom√≠a con desempleo involuntario y capital ocioso y otra con pleno empleo. El nivel de precios se encuentra fijado debido a la subutilizaci√≥n de factores y la ausencia de presi√≥n al alza en los salarios.

En la representaci√≥n de alguna grafica se observa que el punto de intersecci√≥n de las curvas define el nivel de producto de pleno empleo, por lo que impl√≠citamente se tiene que la Oferta Agregada es enteramente inel√°stica a los precios.

Modelo estatico El modelo IS-LM (Inversi√≥n-Ahorro / Liquidez-Dinero) es un modelo macroecon√≥mico que describe el equilibrio simult√°neo en los mercados de bienes y servicios (IS) y en el mercado monetario (LM). Aunque en su forma b√°sica es un modelo est√°tico (es decir, no involucra ecuaciones diferenciales

c0 = 20     # consumo aut√≥nomo

Modelo Dinamico

Las versiones din√°micas del modelo IS-LM s√≠ emplean ecuaciones diferenciales para analizar c√≥mo evoluciona la econom√≠a en el tiempo hacia el equilibrio.

La curva IS representa el equilibrio en el mercado de bienes, donde la producci√≥n (Y) depende del gasto agregado. En la versi√≥n din√°mica, se puede asumir que el nivel de producci√≥n no se ajusta instant√°neamente, sino que sigue una ecuaci√≥n diferencial que describe c√≥mo cambia la producci√≥n en el tiempo.

Modelo din√°mico de la curva IS

La ecuaci√≥n diferencial es:

\[ \frac{dr}{dt} = \alpha \left( a - b r - Y \right) \]

Esta ecuaci√≥n dice que el producto (Y) cambia gradualmente para igualar la demanda agregada.

Code 
#| fig-align: 'center'

# Par√°metros del modelo IS

a = 50     # gasto aut√≥nomo

b = 5      # sensibilidad de inversi√≥n al inter√©s

r = 2       # tasa de inter√©s fija

alpha = 0.5 # velocidad de ajuste del ingreso

 

# Ecuaci√≥n diferencial de la curva IS din√°mica

def is_dinamica(t, Y):

    dYdt = alpha * (a - b * r - Y)

    return dYdt

 

# Condici√≥n inicial y tiempo de simulaci√≥n

Y0 = [50]                # ingreso inicial

t_span = (0, 50)

t_eval = np.linspace(*t_span, 500)

 

# Resolver la ecuaci√≥n diferencial

sol = solve_ivp(is_dinamica, t_span, Y0, t_eval=t_eval)

 

# Graficar resultado

plt.figure(figsize=(8,5))

plt.plot(sol.t, sol.y[0], label='Ingreso Y(t)')

plt.axhline(a - b * r, color='red', linestyle='--', label='Y de equilibrio')

plt.xlabel('Tiempo')

plt.ylabel('Ingreso (Y)')

plt.title('Ajuste din√°mico de la curva IS')

plt.legend()

plt.grid(True)

plt.show()

¬øQu√© muestra este gr√°fico?

El gr√°fico ilustra c√≥mo el ingreso (Y) de una econom√≠a cambia a lo largo del tiempo y se ajusta hacia su nivel de equilibrio de acuerdo con la din√°mica del modelo IS.

El ingreso no se ajusta instant√°neamente, sino gradualmente, hacia el equilibrio. Este ajuste es m√°s r√°pido cuanto mayor es alpha

Curva azul: Representa la trayectoria del ingreso Y(t) a lo largo del tiempo.

Si el ingreso inicial Y = 50 est√° por encima del equilibrio, el ingreso disminuye gradualmente.

Es un modelo √∫til para mostrar c√≥mo una econom√≠a corrige desequilibrios en el corto plazo dentro del marco IS.

El gr√°fico muestra que la econom√≠a comienza en una situaci√≥n de desequilibrio (ingreso diferente de la demanda agregada) y se ajusta gradualmente hasta alcanzar un nuevo equilibrio estable. Este proceso es lento, pero converge, siempre que Œ±>0

Modelo Din√°mico de la Curva LM

La ecuaci√≥n diferencial que describe esta din√°mica es:

\[ \frac{dr}{dt} = \beta (cY - dr - \frac{M}{P}) \]

Code 
#| fig-align: 'center'

# Par√°metros del modelo LM

c = 0.5       # sensibilidad al ingreso

d = 4      # sensibilidad a la tasa de inter√©s

Y = 80      # ingreso constante

M_P = 100    # oferta real de dinero

beta = 0.5  # velocidad de ajuste de la tasa de inter√©s

 

# Ecuaci√≥n diferencial de la curva LM din√°mica

def lm_dinamica(t, r):

    drdt = beta * (c * Y - d * r - M_P)

    return drdt

 

# Condici√≥n inicial y tiempo de simulaci√≥n

r0 = [0.5]              # tasa de inter√©s inicial

t_span = (0, 50)

t_eval = np.linspace(*t_span, 500)

 

# Resolver la ecuaci√≥n diferencial

sol = solve_ivp(lm_dinamica, t_span, r0, t_eval=t_eval)

# Graficar resultado

plt.figure(figsize=(8,5))

plt.plot(sol.t, sol.y[0], label='Tasa de inter√©s r(t)')

plt.axhline((c * Y - M_P) / d, color='red', linestyle='--', label='r de equilibrio')

plt.xlabel('Tiempo')

plt.ylabel('Tasa de inter√©s (r)')

plt.title('Ajuste din√°mico de la curva LM')

plt.legend()

plt.grid(True)

plt.show()

¬øQu√© muestra este gr√°fico?

El gr√°fico muestra c√≥mo la tasa de inter√©s r(t) evoluciona con el tiempo hacia un valor de equilibrio. Este comportamiento es resultado de la interacci√≥n entre la oferta y demanda de dinero. El sistema converge siempre que Œ≤>0, lo que garantiza un proceso de ajuste estable.

C√≥mo la tasa de inter√©s (r) se ajusta con el tiempo para equilibrar el mercado de dinero. El equilibrio se alcanza en: r = (c Y - d r - ) La l√≠nea roja punteada indica ese valor de equilibrio.

An√°lisis econ√≥mico: Este gr√°fico muestra c√≥mo se ajusta la tasa de inter√©s para equilibrar el mercado de dinero.

El sistema converge hacia un valor estable si Œ≤>0, como en este caso. Refleja que si hay m√°s demanda de dinero que oferta, r sube. Si hay m√°s oferta de dinero que demanda,r baja.

Modelo IS-LM Din√°mico Completo

El sistema de ecuaciones diferenciales es:

\[\begin{cases} \frac{dY}{dt} = \alpha \left( a - b r - Y \right) \quad \text{(curva IS din√°mica)} \\ \frac{dr}{dt} = \beta \left( c Y - d r - \frac{M}{P} \right) \quad \text{(curva LM din√°mica)} \end{cases}\]

Este sistema muestra c√≥mo se ajustan el ingreso Y y la tasa de inter√©s r con el tiempo.

Code 
#| fig-align: 'center'
# Par√°metros del modelo

a = 100     # gasto aut√≥nomo

b = 20      # sensibilidad de inversi√≥n al inter√©s

c = 1       # sensibilidad demanda de dinero al ingreso

d = 10      # sensibilidad demanda de dinero al inter√©s

M_P = 50    # oferta real de dinero

alpha = 0.5 # velocidad de ajuste del ingreso

beta = 0.5  # velocidad de ajuste del inter√©s

 

# Sistema IS-LM din√°mico

def is_lm_dinamico(t, z):

    Y, r = z

    dYdt = alpha * (a - b * r - Y)

    drdt = beta * (c * Y - d * r - M_P)

    return [dYdt, drdt]

# Condiciones iniciales y tiempo

z0 = [50, 0.5]              # Y inicial, r inicial

t_span = (0, 20)

t_eval = np.linspace(*t_span, 5000)

 

# Resolver el sistema

sol = solve_ivp(is_lm_dinamico, t_span, z0, t_eval=t_eval)

 

# Extraer soluciones

Y_sol, r_sol = sol.y

 

# Graficar trayectoria en el plano (Y, r)

plt.figure(figsize=(12, 5))

 

plt.subplot(1, 2, 1)

plt.plot(Y_sol, r_sol, color='blue')

plt.xlabel('Ingreso (Y)')

plt.ylabel('Tasa de inter√©s (r)')

plt.title('Trayectoria din√°mica en el espacio (Y, r)')

plt.grid(True)

 

# Graficar evoluci√≥n temporal de Y y r

plt.subplot(1, 2, 2)

plt.plot(sol.t, Y_sol, label='Ingreso Y(t)', color='green')

plt.plot(sol.t, r_sol, label='Tasa de inter√©s r(t)', color='purple')

plt.xlabel('Tiempo')

plt.title('Evoluci√≥n temporal de Y y r')

plt.legend()

plt.grid(True)

 

plt.tight_layout()

plt.show()

Ecuaciones. Usaremos un sistema de ecuaciones diferenciales ordinarias (EDO). Se utiliza para simular c√≥mo evolucionan dos variables clave en una econom√≠a:

Y: el PIB real (nivel de producci√≥n)

i: la tasa de inter√©s

Code 
#| fig-align: 'center'

# Sistema de ecuaciones diferenciales IS-LM
def is_lm_system(y, t):
    Y, i = y
    # Funci√≥n IS: Relaci√≥n entre el PIB y la tasa de inter√©s
    dYdt = alpha * (A - c*Y - d*i)
    # Funci√≥n LM: Relaci√≥n entre la oferta monetaria y la tasa de inter√©s
    didt = beta * (e*Y - f*i - M_real)
    return [dYdt, didt]

Ecuacion y Solucion. Se calcula la matriz Jacobiana que contiene las derivadas parciales de las ecuaciones din√°micas respecto a las variables Y e i.

Code 
#| fig-align: 'center'

# Funci√≥n que resuelve las ecuaciones IS-LM
def is_lm_system(y, t):
    Y, i = y
    dYdt = alpha * (A - c*Y - d*i)  # Ecuaci√≥n de la curva IS
    didt = beta * (e*Y - f*i - M_real)  # Ecuaci√≥n de la curva LM
    return [dYdt, didt]

# Punto de equilibrio (resolver IS y LM)
def punto_equilibrio():
    # Resolviendo las ecuaciones simult√°neas A = cY + d * i y M = e * Y - f * i
    # Para esto, despejamos Y e i
    A_eq = A
    M_eq = M_real
    
    # Resolviendo el sistema
    # IS: A = c * Y + d * i => i = (A - c * Y) / d
    # LM: M = e * Y - f * i => sustituimos i
    def sistema(Y):
        i = (A_eq - c * Y) / d
        return e * Y - f * i - M_eq
    
    # Encontrar el equilibrio resolviendo la ecuaci√≥n para Y
    from scipy.optimize import fsolve
    Y_eq = fsolve(sistema, 10)[0]  # Valor inicial para encontrar la ra√≠z
    
    # Sustituir Y_eq en la ecuaci√≥n de IS para obtener i_eq
    i_eq = (A_eq - c * Y_eq) / d
    return Y_eq, i_eq

Punto de equilibrio. Si los autovalores tienen partes reales negativas, el sistema es estable. Si tienen partes reales positivas, el sistema es inestable.La matriz Jacobiana es una herramienta clave para analizar la estabilidad din√°mica de un sistema de ecuaciones diferenciales. En este contexto, la matriz Jacobiana permite entender c√≥mo las peque√±as variaciones en el sistema afectan al equilibrio.

Esta matriz nos muestra c√≥mo reaccionan las tasas de cambio de Y y i a peque√±as perturbaciones en su entorno.

Code 
#| fig-align: 'center'

# Calcular el punto de equilibrio
Y_eq, i_eq = punto_equilibrio()
print(f"Punto de Equilibrio: PIB = {Y_eq:.2f}, Tasa de inter√©s = {i_eq:.2f}%")

# Derivadas parciales para la matriz Jacobiana
def jacobiana(Y, i):
    dY_dY = -alpha * c     # Parcial de dY/dt respecto a Y
    dY_di = -alpha * d     # Parcial de dY/dt respecto a i
    di_dY = beta * e       # Parcial de di/dt respecto a Y
    di_di = -beta * f      # Parcial de di/dt respecto a i
    J = np.array([[dY_dY, dY_di], [di_dY, di_di]])
    return J

# Calcular la matriz Jacobiana en el punto de equilibrio
J_eq = jacobiana(Y_eq, i_eq)
print("Matriz Jacobiana en el punto de equilibrio:")
print(J_eq)

# Calcular los autovalores de la matriz Jacobiana
autovalores = eigvals(J_eq)
print("Autovalores de la matriz Jacobiana:", autovalores)

# Estabilidad:
if np.all(np.real(autovalores) < 0):
    print("El punto de equilibrio es estable.")
else:
    print("El punto de equilibrio es inestable.")
